# SNAproject10
A study on the changes in online conversation on food relating topics. Main topics include healthy and unhealthy eating and sustainability.

Execution order of codefiles is as follows:
  1. Sisyphus.py to scrape the data with initial keywordlist as a filter.
  2. Temporal_segmenter.py to separate the large code file based on year.
  3. Helpus.ipynb starting from top to (hopefully) get the same results as the report. (Note: Analysis / colorization via Gephi using modularity scores will yield different colors for different communities each time.

Team consists of Ursa M., Petrus H. and Leevi A.
